Plan around weather with saved cities, forecasts, alerts, air quality, and activity recommendations.

## Features

- **City Management** — Save and switch between favorite locations
- **Forecast Display** — Multi-day weather forecasts with details
- **Alert System** — Track weather warnings and advisories
- **Air Quality** — Monitor AQI and health recommendations
- **Activity Planning** — Weather-based activity suggestions
- **Search & Filter** — Find locations and forecasts
- **Analytics View** — Weather pattern distribution
- **Import / Export** — JSON-based location data portability

## Tech Stack

| Layer | Technology |
|-------|-----------|
| Markup | HTML5 |
| Styling | CSS3 (Custom Properties, Grid, Flexbox) |
| Logic | Vanilla JavaScript (ES2022) |
| Storage | LocalStorage |
| Server | Python HTTP server (dev) |

## Getting Started

### Prerequisites

- Python 3.x (for dev server)
- Modern web browser

### Installation

```bash
git clone https://github.com/vizvasanlya/First.git
cd First/vizvasanlya-weather-app
```

### Run

```bash
npm start
```

Open [http://localhost:5173](http://localhost:5173)

### Docker

```bash
docker compose up --build
```
